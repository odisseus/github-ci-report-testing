This is a repository for testing Github workflows.

There are some fake test reports in `sample-reports`. The workflow in `ci.yml` takes these reports and publishes them as if they were generated by tests. The workflow in `test-report-summary.yml` should take these published artifacts and generate an aggregated report of all failed tests.

To help me test these workflows, please create a fork and submit a PR to the `work` branch. You may modify the outcome of "testing" by editing the XML reports, or just edit this file to make an irrelevant change.
